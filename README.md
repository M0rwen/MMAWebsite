# MMA Website

This is a static website for MMA, featuring sections on the home page, solar system, and rotoscoping.

## Theme
- Formal and monotone (black and white with gray accents).

## Setup for GitHub Pages
1. Go to your repository settings on GitHub.
2. Navigate to the "Pages" section.
3. Under "Source", select "Deploy from a branch".
4. Choose the "main" branch and "/ (root)" folder.
5. Save the changes.
6. Your site will be available at `https://m0rwen.github.io/MMAWebsite/`.

## Local Development
Open `index.html` in your browser to view the site locally.
