Rotoscoping

Rotoscoping is an animation technique that involves tracing over live-action footage frame by frame to create realistic animation.

For this project, I used Flipaclip for the frame-by-frame rotoscoping work and Adobe Premiere Pro for video editing and compositing.

Rotoscoping Process in Flipaclip

Here's the step-by-step procedure I followed to create rotoscoped animation with color in Flipaclip:

  1. Import Reference Video: Open Flipaclip and create a new project. Import your reference video using the import tool. Set the project frame rate to 30 fps to match the reference footage.
  2. Set Up Animation Layers: Create a new layer above the reference video layer. Flipaclip automatically handles layer management for frame-by-frame animation.
  3. Enable Onion Skinning: Turn on onion skinning in the settings (usually found in the layer or animation menu) to see previous and next frames as semi-transparent overlays, helping maintain consistency in movement.
  4. Choose Drawing Tools: Select the brush tool with appropriate size and opacity. Flipaclip offers various brush types optimized for touch drawing on mobile devices.
  5. Start Tracing: For each frame, carefully trace the subject using a bright, contrasting color (like white or bright blue) on a transparent layer. Use the frame-by-frame playback to ensure smooth motion.
  6. Add Color: Create a new layer above the line art. Use the fill bucket tool or brush to add base colors to different parts of the subject. Use the color picker to sample colors from the reference frame.
  7. Refine Details: Add shading, highlights, and textures using the brush tools. Flipaclip supports layer blending modes for creating depth and dimension.
  8. Frame-by-Frame Animation: Use the timeline scrubber to navigate between frames. Make small adjustments to maintain smooth motion and consistent timing at 30 fps.
  9. Export Animation: Export the completed animation as an MP4 file or image sequence directly from Flipaclip's export menu.
  10. Composite in Premiere Pro: Import the rotoscoped animation into Adobe Premiere Pro and composite it over the original footage for the final result.
